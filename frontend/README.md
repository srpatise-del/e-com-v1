# CameraHub TH

เว็บ E-Commerce ขายกล้องดิจิทัลภาษาไทย แยกเป็น frontend ด้วย Vite + React และ backend ด้วย Express + MongoDB

## Frontend

```bash
npm install
npm run dev
```

สร้างไฟล์ `.env` จาก `.env.example`

## Backend

```bash
cd backend
npm install
npm run server
```

สร้างไฟล์ `backend/.env` จาก `backend/.env.example`

## Seed Data

```bash
cd backend
npm run seed
```
